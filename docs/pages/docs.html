Project Overview

This project is a comprehensive demonstration of integrating Microsoft Graph API into a web application. It consists of two main components:

1. Python Backend (FastAPI)

A robust backend API built with FastAPI that handles:

  • OAuth 2.0 Authorization Code Flow
  • Token management (MSAL)
  • Direct Graph API calls using HTTPX
  • Modular architecture

2. Static Frontend (Demo Site)

This interactive documentation site hosted on GitHub Pages that:

  • Simulates API responses (Mock API)
  • Demonstrates UI/UX patterns
  • Explains data structures
  • Requires no backend to run

Architecture

The backend follows a modular layered architecture to ensure separation of concerns and maintainability. 

src/
├── api/          # Route handlers (Controllers)
├── core/         # Configuration & Security (Auth)
├── models/       # Pydantic models (Data Schemas)
├── services/     # Business logic & Graph Client
└── main.py       # Application entry point

Setup & Installation

Prerequisites

  • Python 3.11+
  • Docker (optional)
  • Azure AD Application Registration

Local Development

1. Clone the repository:

git clone https://github.com/270803ggiztheking-rgb/ms-graph-api-demo.git

2. Create virtual environment and install dependencies:

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

3. Configure Environment Variables (.env):

TENANT_ID=your_tenant_id
CLIENT_ID=your_client_id
CLIENT_SECRET=your_client_secret

4. Run the application:

uvicorn src.main:app --reload
